# tg_bot
Программа tg_bot, реализованная в боте iToora_bot, 
позволяет работать 
с калькулятором в боте. 
Расчет производится вызовом команды /calc  
с последущим выражением для расчета:
например, `/calc 25*4/20 +5*(2+2)`
